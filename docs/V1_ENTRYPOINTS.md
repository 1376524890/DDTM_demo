# V1 executable entrypoints

- `ddtm_zkp/cmd/v1setup`: compile circuits, setup keys, export verifiers.
- `ddtm_zkp/cmd/v1prove`: commitments and proof CLI.
- `ddtm_zkp/cmd/v1prover`: authenticated HTTP proof service.
- `ddtm_evm/scripts/deploy_v1.js`: deploy verifiers and DDTM contract.
- `services/gateway/src/server.js`: integrated protocol API and indexer.
- `scripts/v1_smoke.sh`: complete buyer/seller execution test.
