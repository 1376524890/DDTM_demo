# DDTM V1 reference prototype

DDTM V1 is the Hardhat/MinIO reference implementation used to demonstrate the paper protocol as a complete executable workflow.

## Start

```bash
cp .env.example .env
docker compose up --build -d
docker compose ps
make smoke
```

The stack contains a local Hardhat EVM, three generated gnark Groth16 verifier contracts, the DDTM escrow contract, a Go prover, PostgreSQL, MinIO and the Node.js protocol gateway.

## Test without Docker

```bash
make test-v1
```

This command regenerates circuit artifacts and Solidity verifier contracts, then runs:

- circuit relation tests;
- contract state-machine and timeout tests using configurable verifier mocks;
- real Groth16 proofs generated by gnark and verified by the exported Solidity contracts on Hardhat;
- gateway cryptography, authentication and prover-protocol tests.

See `docs/V1_SYSTEM_DESIGN.md` for trust boundaries and `docs/API.md` for the HTTP interface.
