# DDTM V1 security scope

DDTM V1 is a research artifact. It demonstrates protocol execution, proof verification, storage binding, escrow exits and evidence-bound dispute paths. It has not received an independent smart-contract audit and must not hold real assets.

The local API key, shared HMAC secret, database master key and deterministic Hardhat mnemonic are development credentials. A production deployment requires secret rotation, mTLS, KMS/HSM-backed keys, organizational authorization, monitored backups and an audited upgrade/governance procedure.
